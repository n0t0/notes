### Linux Keyboard Shortcuts
```
Ctrl - A 		—> beginning of command line 
Ctrl - E 		—> end of command line
Esc - B		    —> back one word 
Esc - F 		—> forward one word
Ctrl - B 		—> back one character 
Ctrl - F 		—> forward one character
SPACEBAR 	    —> page down
esc + v 		—> page up


Ctrl - D 		—> delete a character to the right 
Backspace 	    —> remove a character to the left
Ctrl - R 		—> redisplay current command line 
Ctrl - U 		—> Erase a line 
ctrl + k        —> Erase to the right
ctrl + x        —> Erase to the left 


Ctrl - Z 		—> ends configuration mode and returns to privileged EXEC mode
Tab 			—> complete command
Ctrl - P 		—> up arrows
Ctrl - N 		—> down arrow 
Ctrl - T 		—> reverse characters


esc + u         —> converts to uppercase
esc + l         —> converts to lowercase 
esc + c         —> first letter uppercase 

ctrl + x,ctrl + e -> calls a default editor 

!! —> retrieve last command
```
